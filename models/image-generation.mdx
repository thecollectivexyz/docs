## Overview

NTS Vortex can generate images using AI. This feature lets users create visuals for presentations, marketing materials, documentation, and more—all through simple text descriptions.

## Available Image Generation Models

Common image generation models include:

| Model | Provider | Strengths |
|-------|----------|-----------|
| DALL-E 3 | OpenAI | High quality, follows prompts well |
| DALL-E 2 | OpenAI | Good quality, lower cost |
| Stable Diffusion | Various | Versatile, many options |
| Midjourney | Midjourney | Artistic, creative styles |

<Info>
Available models depend on your configured providers and API connections.
</Info>

## Enabling Image Generation

<Steps>
  <Step title="Access Admin Settings">
    Go to **Settings** → **Admin Panel** → **Settings**.

    <Frame caption="Admin settings path">
      <img src="/images/placeholders/models/admin-image-settings.png" alt="Path to admin settings" />
    </Frame>
  </Step>

  <Step title="Find Image Settings">
    Look for **Images**, **Image Generation**, or **Media** settings.

    <Frame caption="Image settings section">
      <img src="/images/placeholders/models/image-settings-section.png" alt="Image generation settings section" />
    </Frame>
  </Step>

  <Step title="Enable image generation">
    Toggle the **Enable Image Generation** option on.

    <Frame caption="Enable toggle">
      <img src="/images/placeholders/models/enable-image-gen.png" alt="Toggle to enable image generation" />
    </Frame>
  </Step>

  <Step title="Configure the provider">
    Select your image generation provider and enter any required API keys.

    <Frame caption="Provider configuration">
      <img src="/images/placeholders/models/image-provider-config.png" alt="Image provider configuration fields" />
    </Frame>
  </Step>

  <Step title="Save settings">
    Click **Save** to apply your configuration.
  </Step>
</Steps>

## Image Generation Best Practices

<AccordionGroup>
  <Accordion title="Writing effective prompts">
    Good prompts lead to better images:
    
    **Basic prompt:**
    > "A truck"
    
    **Better prompt:**
    > "A modern red semi-truck driving on a desert highway at golden hour, professional photography style, high detail"
    
    Include:
    - Subject description
    - Setting/background
    - Style or mood
    - Quality descriptors
  </Accordion>

  <Accordion title="Professional use tips">
    For business materials:
    - Specify "professional" or "corporate" style
    - Request "clean" or "minimalist" designs
    - Mention intended use (presentation, report, etc.)
    - Ask for specific color schemes if needed
  </Accordion>

  <Accordion title="Iterating on results">
    If the first result isn't quite right:
    - Ask for modifications: "Make it more vibrant"
    - Request variations: "Show me 3 different versions"
    - Be more specific: "Change the background to blue"
  </Accordion>
</AccordionGroup>

## Content Guidelines

<Warning>
AI image generation should be used responsibly. Ensure generated images:
- Are appropriate for workplace use
- Don't infringe on trademarks or copyrights
- Align with NTS brand and values
</Warning>

Most AI image generators have content filters that prevent:
- Inappropriate content
- Copyrighted characters/logos
- Real people's likenesses

## Troubleshooting

<AccordionGroup>
  <Accordion title="Image generation not working">
    - Verify image generation is enabled in settings
    - Check that the API key is valid
    - Ensure you have credits/quota remaining
    - Try a simpler prompt
  </Accordion>

  <Accordion title="Images are low quality">
    - Check quality settings (switch to HD if available)
    - Use larger image sizes
    - Write more detailed prompts
    - Try a different model
  </Accordion>

  <Accordion title="Request rejected">
    - The prompt may have triggered content filters
    - Try rephrasing the request
    - Remove any potentially problematic terms
  </Accordion>

  <Accordion title="Generation takes too long">
    - Higher quality settings take longer
    - Larger sizes take longer
    - Try standard quality for faster results
    - Check your internet connection
  </Accordion>
</AccordionGroup>
