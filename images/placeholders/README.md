## Screenshot Guidelines

1. **Resolution**: Capture at 1920x1080 or higher
2. **Format**: PNG format preferred
3. **Naming**: Use the exact filenames listed above
4. **Content**: Blur or redact any sensitive information
5. **Consistency**: Try to maintain consistent window sizes and themes

## Adding Screenshots

1. Take the screenshot following the guidelines
2. Save it to the appropriate subdirectory
3. The documentation will automatically display it
