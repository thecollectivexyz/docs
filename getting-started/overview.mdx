## Overview

NTS Vortex is Nationwide Transport Services' custom AI assistant platform. Built to help your team work smarter, NTS Vortex provides access to powerful AI models that can assist with a wide range of tasks—from answering questions about company policies to helping draft communications and analyzing documents.

## Who Should Use NTS Vortex?

NTS Vortex is designed for all Nationwide Transport Services team members who need AI assistance in their daily work. As an administrator, you have additional capabilities to:

- **Configure system settings** - Manage environment variables and server configuration
- **Create and manage agents** - Build specialized AI assistants for different departments
- **Manage the knowledge base** - Upload and organize documents for AI reference
- **Monitor usage** - View chat history and user activity
- **Control model access** - Select which AI models are available to users

## How NTS Vortex Works

<Steps>
  <Step title="User asks a question">
    A team member types a question or request in the chat interface.
  </Step>
  <Step title="AI processes the request">
    NTS Vortex sends the question to the selected AI model, along with any relevant context from the knowledge base.
  </Step>
  <Step title="AI generates a response">
    The AI model analyzes the question and generates a helpful response.
  </Step>
  <Step title="User receives the answer">
    The response appears in the chat, and the user can continue the conversation or start a new one.
  </Step>
</Steps>

## System Requirements

NTS Vortex runs on a dedicated Hostinger VPS (Virtual Private Server). As an administrator, you may occasionally need to:

- Access the server to perform updates
- Modify configuration settings
- Monitor system health
